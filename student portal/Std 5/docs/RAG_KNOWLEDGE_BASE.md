# 🧠 RAG Knowledge Base — Class 5 AI Buddy

> **RAG = Retrieval-Augmented Generation** — The AI Buddy searches this knowledge base before answering, so responses are grounded in actual Class 5 curriculum content rather than guesses.

---

## How It Works

```
Student asks a question
        ↓
AI searches INITIAL_KNOWLEDGE (vector similarity)
        ↓
Top matching chunks are injected into the prompt
        ↓
Groq LLM answers using the retrieved context
```

**Entry point:** `constants.tsx`
```ts
export const INITIAL_KNOWLEDGE: TextbookChunk[] = [
  ...PDF_KNOWLEDGE_BASE,   // data/knowledgeChunks.ts  — PDF-extracted chunks
  ...QA_KNOWLEDGE_BASE,    // data/qaData.ts           — hand-curated Q&A pairs
];
```

## Adding New Q&A Entries

Open `data/qaData.ts` and append to `ENGLISH_QA` or `MATHS_QA` following this format:

```ts
{
  id: 'qa-en-51',          // next sequential ID
  subject: 'English',      // 'English' or 'Maths'
  page: 1,                 // page number (use 1 for grammar topics)
  chapter: 'Grammar – Tenses',   // topic label (used for retrieval grouping)
  content: 'Q: Your question here?\nA: Your answer here.',
},
```

Then make sure `QA_KNOWLEDGE_BASE` at the bottom of the file includes the array with your new entry.
