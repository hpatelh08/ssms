## Quick Tips for Image Generation:

1. **DALL-E 3 (Free via Bing)**
   - Go to: https://www.bing.com/images/create
   - Copy each prompt above
   - Download PNG images
   - Resize if needed using online tools

2. **Important Notes:**
   - Always specify "transparent background" for objects (NOT background)
   - Mention "PNG" in prompt for better quality
   - Use "centered" to ensure object is in middle
   - If results aren't consistent, regenerate with same prompt

3. **After Generation:**
   - Save each file with exact names above
   - Place in: `public/game/`
   - Verify transparency (except ocean_background.png)

4. **Free Alternative:**
   - Visit Kenney.nl and download their underwater/ocean asset packs
   - They're free, no attribution required, perfect for prototypes
