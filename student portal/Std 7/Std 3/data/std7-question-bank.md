## Game 2: Find the Mistake

Q1. (Maths - Integers - EASY) -8 + 3 = 11
Mistake: Sign rule for integers is used incorrectly.
Correction: -8 + 3 = -5
Explanation: Adding 3 to -8 moves 3 steps toward zero.

Q2. (Maths - Fractions - EASY) 5/8 is greater than 3/4.
Mistake: Fraction comparison is wrong.
Correction: 5/8 is less than 3/4.
Explanation: 3/4 is equal to 6/8, and 5/8 is smaller.

Q3. (Maths - Percentages - MEDIUM) 25% of 160 is 25.

Correction: 25% of 160 is 40.
Explanation: 25% means one-fourth of the number.

Q4. (Maths - Algebra - MEDIUM) If x + 9 = 14, then x = 23.

Correction: If x + 9 = 14, then x = 5.
Explanation: Subtract 9 from both sides.

Q5. (Maths - Fractions - MEDIUM) 3/5 + 1/5 = 4/10
Mistake: Denominator is changed incorrectly.
Correction: 3/5 + 1/5 = 4/5
Explanation: When denominators are same, only numerators are added.
