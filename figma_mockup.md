# Smart School Management System - Figma Mockup Description

## Overview
This document describes the Figma mockup design for the Smart School Management System. The design focuses on a clean, intuitive, and responsive interface for all user types.

## Color Palette
- Primary: #667eea (Blue-Purple gradient)
- Secondary: #764ba2
- Success: #4facfe to #00f2fe
- Warning: #fad961 to #f76b1c
- Danger: #ff6b6b to #ee5a24
- Info: #0fd850 to #f9f047

## Layout Structure

### 1. Login Page
- **Header**: School logo with "Smart School System" title
- **Portal Selector**: Three tabs - Admin, Student, Parent
- **Admin Options**: Sub-options for HOD and Teacher when Admin is selected
- **Form Fields**: Username, Password inputs with icons
- **Background**: Animated gradient with subtle particles

### 2. Dashboard Layout
- **Sidebar**: Fixed navigation menu with icons and labels
- **Main Content**: Responsive grid layout with cards
- **Header**: Page title and user profile section
- **Statistics Section**: Key metrics displayed in cards

### 3. Card Design
- **Rounded corners**: 15px radius
- **Shadow**: Subtle drop shadow with hover effect
- **Icons**: Colored background with white icons
- **Hover Effect**: Lift effect with increased shadow

## Page-Specific Designs

### Login Page
- Centered card layout
- Smooth animations on load
- Tab-based navigation for portals
- Clear visual hierarchy

### HOD Dashboard
- 3 main feature cards: Exam Timetable, Teacher Progress, Class Progress
- Additional cards for Academic Reports, Student Management, Staff Management
- Statistics panel with key metrics

### Teacher Dashboard
- 6 main feature cards covering all teacher needs
- Profile, Attendance, Homework, Marks, Assignments, Files
- Statistics panel for class-related metrics

### Student Dashboard
- 3 main feature cards: Homework, Exams, Overall Performance
- Modal popup for detailed performance view
- Statistics panel with personal metrics

### Parent Dashboard
- 12 feature cards covering all parent needs
- Comprehensive statistics panel
- Clean categorization of information

## Responsive Design
- Mobile-friendly sidebar toggle
- Grid layout adjusts based on screen size
- Touch-friendly elements
- Readable typography on all devices

## Animations
- Fade-in effects for content
- Hover animations on interactive elements
- Smooth transitions between states
- Loading animations where appropriate

## Typography
- Font: Poppins family
- Headers: Bold weights for emphasis
- Body text: Regular weights for readability
- Consistent sizing hierarchy

## Navigation
- Intuitive sidebar navigation
- Clear visual indication of current page
- Consistent iconography
- Smooth page transitions
